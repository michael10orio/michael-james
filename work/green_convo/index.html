Green Convo

The Green Convo application is a simple chat application that let users create an account, see the list of users and message them.This application provides a communication platform for users to connect and conversate on any device. For the designs, I choose friendly colors such as varieties of green. I put big forms and large font sizes on both messages and user lists for the users to navigate easily. For this project, I worked individually. I enjoyed learning new essential codes, designing the interface, and how everything turned out with my Green Convo app.
